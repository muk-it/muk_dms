MuK Documents Access

Documents Access Control

MuK IT GmbH - www.mukit.at

Overview

MuK Documents Access extends the document module by an extended access control. It allows you to create and manage access groups. These groups can be used to specify access to documents.

Access Groups

Groups can be set to any directory or file. The users of a group are defined by a job or a given Odoo group.

Group Inheritance

Groups are inherited by any subdirectory or file. In case this is not the wanted behavior the mechanisms can be turned off by unchecking the checkbox.
