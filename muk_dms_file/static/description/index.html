MuK Documents File

Documents File Storage

MuK IT GmbH - www.mukit.at

Overview

MuK Documents is a module to create, manage and view files directly within Odoo. MuK Documents File enables a new save option. Files saved with this option will be stored inside a file storage.

File Storage

Save files to a file store instead of your database.

File Data

A new view is provided to see all stored files. If a file is edited outside of Odoo it is necessary to update its checksum.
