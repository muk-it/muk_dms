MuK Documents Thumbnails

Automatic File Thumbnails

MuK IT GmbH - www.mukit.at

Overview

MuK Documents Thumbnails allows the creation of thumbnails for files. Thumbnails can be created either directly when changing the file or via a cron job once per hour to avoid interrupting the workflow unnecessarily.

Settings

Thumbnails can be created either directly when changing the file or via a cron job once per hour. The cron job runs in the background and therefore does not interfere with the user workflow.

Configuration

The thumbnail service supports a variety of file types, but for some external programs are used and it may be necessary to install them separately. An example of this is the corresponding video codecs. If you want to exclude certain file types from the automatic thumbnail creation you can enter them in the settings.
